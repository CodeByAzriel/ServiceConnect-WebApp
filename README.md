# ServiceConnect-WebApp
While researching the local economy, I noticed that a significant number of young people are unemployed, especially those with skills in tech, tutoring, home services, or creative trades. Many of them don’t have a simple way to connect with clients or showcase their abilities.

ServiceConnect is my solution:

✅ It allows skilled youth to register as service providers.

✅ Clients can easily book services in their city.

✅ Admin oversight ensures safe and trustworthy interactions.

I built this platform to bridge the gap between opportunity and talent, giving young people a chance to earn income while providing real value to the community.

🌟 Key Features

User Registration & Login 🔑

Users can register as service providers or clients.

Admin account for managing the platform (dylan@serviceconnect.com / Admin@2026).

Service Listings & Booking 📋

Providers can add, edit, and delete services.

Clients can view, filter by category or city, and book services.

Bookings can be accepted, declined, or marked complete.

Dashboard & Management 📊

Providers can see pending bookings, recent requests, and manage their services.

Admin can oversee users, services, and reported issues.

Messaging System 💬

Logged-in users can message each other.

Search for users, with restrictions for admin accounts.

Messages display latest conversations first.

Ratings & Reviews ⭐

Clients can leave ratings for services after completion.

Providers can track their performance.

Reporting & Moderation 🚨

Users can report inappropriate content or issues.

Admin receives and manages reports.

🛠 Technology Stack

Frontend: HTML, CSS (Modern UI v7), JavaScript

Backend: PHP, MySQL
